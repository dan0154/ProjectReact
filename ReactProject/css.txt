Css inline:
    -No próprio texto, para aplicações mais simples

CSS regular:
    -Folhas de estilo;
    -Módulos CSS;
    -Pré-processadores: SASS, SCSS, LESS. Permitem criar variáveis;
    -Styled-components;
    -Framweworks;

---------------------------------------------------

SASS: npm install -D SASS

Styled-components: npm install styled-components --save
    -lib do react/react native que permite criar componentes de estilo, com CSS dentro de JS

Framweworks: o mais conhecido é o tailwind (npm install -D tailwind)
    -Fornece classes CSS pré-definidas

---------------------------------------------------

Responsividade:
    -Sistema capaz de se adaptar a diferentes tamanhos de tela e dispositivos
    -Media query: permite qua renderização se adapte a condições como tamanhos de tela
    -Flexbox: modelo de layout CSS que permite criar layouts flexíveis (começa o display: flex).
